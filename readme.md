Projet en développement

Nom
foad_resavelo

Description:
Ce projet consiste à créer un dashboard qui va afficher des données que l'on va pouvoir créer, lire, modifier ou supprimer.
Ce dashboard est rattaché à une base de donnée fonctionelle qui récupère et renvoie des données côté client.

Installation:
J'ai installé wamp et créé mon projet dans le dossier www de wamp64.
Puis j'ai créé un virtual host du nom de resavelo pour héberger le projet

Support

Feuille de route

après avoir créé le projet et définis l'arborescence de mes fichiers, je débute le code
